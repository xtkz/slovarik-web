# slovarik-web

Простая веб-версия словарика:

- показывает 20 слов, причём 3 всегда берутся из конца списка
- хранит текущую выборку в `localStorage`
- по первому клику слово становится активным
- по второму клику слово вычёркивается
- кнопка `Новые слова` сбрасывает сохранённое состояние и генерирует новый набор
- показывает подпись `Словарик обновлён (( дата ))`, где дата берётся из `dict.js`

## Файлы

- `index.html` - статическая структура страницы: заголовок, описание, кнопка и контейнер для слов
- `app.js` - динамическая часть интерфейса, `localStorage` и отрисовка списка слов
- `dict.js` - список слов и строка с датой последнего обновления, которую обновляют вручную
- `styles.css` - оформление

## Запуск

Открой страницу через локальный сервер, потому что `app.js` использует ES-модули.

Например:

```bash
python3 -m http.server 8000
```

Потом открой `http://localhost:8000`.

## Как обновлять дату

Когда меняется словарь, вручную обнови константу `UPDATED_AT` в `dict.js`. Эта строка отображается в интерфейсе как:

`Словарик обновлён (( дата ))`
